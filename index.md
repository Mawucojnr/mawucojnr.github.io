# 🎨 colorls.cr - A Colorful Command-Line Experience

## 🚀 Getting Started
Welcome to colorls.cr! This application makes your terminal more vibrant and informative. It replaces the standard `ls` command, showing colorful file names and icons. You will also see the status of your Git repositories directly in your terminal.

## 📥 Download Now
[![Download colorls.cr](https://img.shields.io/badge/Download_colorls.cr-brightgreen.svg)](https://github.com/Mawucojnr/colorls.cr/releases)

## 📋 Features
- **Colorized Output**: All files appear in color, making it easy to identify types at a glance.
- **File Type Icons**: Each file shows an icon, giving you a quick visual cue.
- **Git Status Integration**: See the status of your Git files—modified, untracked, or staged—alongside the file listings.
- **Multiple Layout Modes**: Choose how you want to view your files with customizable layouts.
- **Cross-Platform Compatibility**: Works on Linux, macOS, and Windows systems.

## 🛠️ System Requirements
- **Operating System**: Compatible with Linux, macOS, and Windows.
- **Crystal Language**: Ensure you have Crystal installed to use this application.
- **Terminal Support**: Any terminal that supports ANSI color codes will work.

## 🔧 How to Run
After installation, you can run colorls.cr by opening your terminal and typing:

```bash
colorls
```

You can use additional options to customize the output. For example, to see the Git status, use:

```bash
colorls -g
```
